# OI-Note
## 部署
下载 Python。

将本仓库 clone 到本地，下载 MkDocs 并启动。
```bash
pip install mkdocs
pip install mkdocs-material
git clone https://github.com/jreeys-man/OI-Note.git
cd OI-Note
mkdocs serve
```

然后就可以在 [127.0.0.1:8000](127.0.0.1:8000) 看见部署的 **OI-Note** 了。

## 贡献

请确保在提交贡献前，修改的页面能正常在自行部署的 **OI-Note** 中正常工作，

### 方法一
在 Issues 中提出建议，由其他人进行修改。

### 方法二
fork 本仓库，在自己的仓库进行修改后，提交 PR（Pull Request）。
